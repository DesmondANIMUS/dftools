# Exiftool

Exiftool is a command line tool  used for reading and writing metadata in many file [formats](https://www.sno.phy.queensu.ca/~phil/exiftool/#supported). It's written in Perl so it's platform-independent and can be used also as an library. 

[Official Website](https://www.sno.phy.queensu.ca/~phil/exiftool/)
